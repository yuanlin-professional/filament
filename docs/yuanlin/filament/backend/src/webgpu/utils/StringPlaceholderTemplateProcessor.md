# StringPlaceholderTemplateProcessor

## 文件概述

字符串模板占位符处理器，用于在 WGSL 着色器模板中查找并替换占位符。支持多遍替换（未匹配的占位符保留原样）。

## 核心函数

- **`processPlaceholderTemplate`** - 主函数
  - 参数: 模板字符串、占位符前缀（如 `{{}}`）、占位符后缀（如 `}}`）、占位符名到值的映射
  - 返回: 替换后的字符串

## 关键实现逻辑

1. **顺序扫描**: 从头到尾扫描模板字符串，遇到前缀时提取占位符名称
2. **查找替换**: 在映射表中查找占位符名，找到则替换，否则保留原始占位符（支持多遍处理）
3. **流式构建**: 使用 `std::stringstream` 逐段拼接输出字符串

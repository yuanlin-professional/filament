# utils (着色器工具)

## 文件概述

`utils/` 目录包含被多个材质共享的着色器工具函数片段。

## 文件

| 文件 | 说明 |
|------|------|
| `depthUtils.fs` | 深度值相关的工具函数（线性化、重建世界位置等） |
| `geometry.fs` | 几何运算工具函数（法线重建、UV 计算等） |

## 关键实现逻辑

- `depthUtils.fs`: 提供深度缓冲值到线性深度/视空间位置/世界空间位置的转换函数。这些函数被 SSAO、DOF、雾效等多个后处理效果共享。
- `geometry.fs`: 提供从深度缓冲重建法线、UV 坐标变换等几何工具。

## 依赖关系

- `frameUniforms` - 用于投影矩阵逆变换等

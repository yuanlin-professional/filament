# clearDepth

## 文件概述

`clearDepth.mat` 定义了深度清除材质，用于将深度缓冲写入最远值（1.0）而不影响颜色附件。

## 材质属性

- **着色模型**: `unlit`
- **顶点域**: `device`
- **颜色写入**: 禁用
- **深度写入**: 启用
- **深度裁剪**: 禁用
- **裁剪**: 无
- **变体过滤**: 排除 skinning、shadowReceiver、vsm、fog

## 着色器逻辑

无自定义着色器代码。使用默认行为渲染一个全屏三角形，仅写入深度值以清除深度缓冲。

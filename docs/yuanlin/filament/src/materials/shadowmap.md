# shadowmap

## 文件概述

`shadowmap.mat` 定义了阴影贴图可视化调试材质，用于在屏幕上渲染阴影贴图的内容以辅助调试。

## 材质属性

- **名称**: shadowmap
- **域**: `postprocess`（后处理）
- **顶点域**: `device`
- **深度写入/裁剪**: 均禁用

## 参数

| 名称 | 类型 | 说明 |
|------|------|------|
| `shadowmap` | sampler2dArray | 阴影贴图数组纹理（不可过滤） |
| `power` | float | 对比度幂值 |
| `scale` | float2 | 视口缩放 |
| `layer` | int | 阴影贴图层索引 |
| `level` | int | mip 级别 |
| `channel` | int | 颜色通道索引 |

## 着色器逻辑

- **顶点着色器**: 将 UV 坐标传递到片段着色器，并按 `scale` 参数缩放位置。
- **片段着色器**: 从阴影贴图数组纹理的指定层和级别采样，取指定通道值，应用幂函数增强对比度后输出为灰度图。

## 依赖关系

- 无额外依赖（纯后处理材质）

# Fog (雾效)

## 文件概述

`fog/` 目录包含屏幕空间雾效的实现（`fog.h`/`fog.cpp`/`fog.mat`），支持距离雾、高度雾和大气散射效果。

## 核心类/结构体

- `fog.h`/`fog.cpp` 提供雾效的 C++ 端参数配置。
- `fog.mat` 定义雾效后处理材质。

## 关键实现逻辑

- **距离雾**: 基于视空间深度计算雾浓度衰减。
- **高度雾**: 基于世界空间高度计算雾密度，低处浓厚、高处稀薄。
- **颜色计算**: 雾颜色可来自常数、IBL 环境光或大气散射模型。
- 通过 `FogOptions` 配置各种参数（密度、起始距离、高度衰减等）。

## 依赖关系

- 深度缓冲 - 用于距离计算
- IBL / `FIndirectLight` - 雾颜色来源
- `FogOptions` - 配置选项

# skybox

## 文件概述

`skybox.mat` 定义了天空盒渲染材质，支持立方体贴图天空盒、纯色模式和太阳光晕效果。

## 材质属性

- **名称**: Skybox
- **着色模型**: `unlit`（无光照）
- **顶点域**: `device`（设备空间）
- **深度写入**: 禁用
- **裁剪**: 无
- **变体过滤**: 排除 skinning、shadowReceiver、vsm

## 参数

| 名称 | 类型 | 说明 |
|------|------|------|
| `showSun` | int | 是否显示太阳光晕 |
| `constantColor` | int | 是否使用纯色模式 |
| `skybox` | samplerCubemap | 天空盒立方体贴图 |
| `color` | float4 | 纯色模式的颜色 |

## 着色器逻辑

- **顶点着色器**: 将裁剪空间坐标逆投影到世界空间，计算视线方向（不除以 w 以避免远距离精度问题）。
- **片段着色器**: 纯色模式直接输出 `color`；否则采样天空盒贴图并乘以 IBL 亮度。Feature Level > 0 时支持太阳光晕，基于视线方向和光源方向的余弦值，通过幂函数衰减生成光晕渐变。

## 依赖关系

- `frameUniforms` - 帧级 uniform（IBL 亮度、光源方向/颜色/强度、太阳参数等）

# blitDepth

## 文件概述

`blitDepth.mat` 定义了深度缓冲复制材质，将深度纹理输出到深度附件。

## 材质属性

- **域**: `postprocess`
- **深度写入**: 启用
- **深度裁剪**: 禁用
- **裁剪**: 无
- **输出**: `depth` 目标为深度附件

## 参数

| 名称 | 类型 | 说明 |
|------|------|------|
| `depth` | sampler2d | 源深度纹理（不可过滤） |
| `viewport` | float4 | 视口参数 |

## 着色器逻辑

从源深度纹理采样红色通道值，直接写入输出深度。用于在渲染目标之间复制深度信息。

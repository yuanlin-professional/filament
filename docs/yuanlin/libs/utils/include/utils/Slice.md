# Slice.h

## 文件概述
固定大小的容器视图，类似 std::span，提供对连续内存区域的非拥有引用。

## 核心类/结构体/函数
- **Slice<T>**: 切片模板类
  - 构造自 begin/end 指针对
  - 提供完整的随机访问迭代器接口
  - `size()`, `empty()`, `data()`: 基本查询
  - `ssize()`: 有符号大小
  - `hash<Hash>()`: 计算内容哈希

## 关键实现逻辑
- 零开销抽象，仅存储两个指针
- 比较操作逐元素进行

## 依赖关系
- `utils/Hash.h`, `utils/compiler.h`

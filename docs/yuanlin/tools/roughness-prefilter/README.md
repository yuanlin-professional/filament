# roughness-prefilter - 粗糙度预滤波贴图生成工具

## 模块名称和概述

`roughness-prefilter` 是一个用于生成预滤波粗糙度贴图的命令行工具。它根据输入的法线贴图计算出经过预滤波处理的粗糙度值，输入粗糙度可以是常数或粗糙度贴图。生成的预滤波粗糙度贴图可在运行时用于减少着色走样（shading aliasing），提高远距离渲染质量。

## 核心功能

- **法线贴图分析**: 读取法线贴图并分析法线分布的方差
- **粗糙度预滤波**: 基于法线分布计算各像素的有效粗糙度
- **常数/贴图输入**: 输入粗糙度可以是固定值或粗糙度纹理
- **减少走样**: 通过将法线细节信息编码到粗糙度中，避免远距离渲染时的高光闪烁
- **多格式支持**: 通过 imageio 库支持 PNG、HDR、EXR 等图像格式

### 使用方式

```bash
# 基于法线贴图生成预滤波粗糙度贴图
roughness-prefilter [options] <normal_map> <output>
```

### 原理说明

在 PBR 渲染中，当相机距离物体较远时，一个像素可能覆盖多个纹素的法线。如果直接采样法线贴图进行着色，会产生高光闪烁（specular aliasing）。预滤波粗糙度通过分析法线贴图的局部方差，将高频法线细节转化为增加的粗糙度值，从而在 mipmap 较高层级时自动平滑高光响应。

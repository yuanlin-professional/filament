# main.cpp (rgb-to-lmsr)

## 文件概述
RGB 到 LMSR 转换矩阵计算工具。通过光谱积分计算 Rec.709 RGB 色彩空间到人眼视锥细胞（LMS）和视杆细胞（R）响应的 4x3 转换矩阵，用于色觉模拟等场景。

## 核心函数
- `main()` - 读取光谱数据 -> 执行积分 -> 输出矩阵

## 关键实现逻辑
1. 使用 CIE 1931 2 度观察者 Y 亮度匹配函数和 D65 标准光源光谱功率分布进行归一化
2. Rec.709 反射光谱（401 个采样点，390-790nm）来自 Scott Burns 的快速 RGB 到光谱转换
3. 锥体/杆体响应数据来自 Stockman & Sharpe (2000) 2 度视锥基础函数和 CIE (1951) 暗视觉函数
4. 对每个 RGB 通道，通过光谱加权求和计算 L/M/S/R 响应值
5. `--normalize` 选项使用 D65 光源加权归一化

## 依赖关系
- `math/` - 矩阵和向量运算

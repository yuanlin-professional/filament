# main.cpp (specular-color)

## 文件概述
导体镜面反射颜色计算工具。从光谱折射率数据（复数 IOR: n+ik）计算金属材质在正入射（f0）和指定角度（默认 81.7 度）下的 sRGB 基础颜色，用于 PBR 材质参数设置。

## 核心函数
- `main()` - 入口：读取光谱数据文件 -> 计算颜色 -> 输出 linear/sRGB/hex 颜色值
- `computeColor()` - 对可见光谱积分计算 XYZ 颜色，转换为 sRGB
- `fresnel()` - 复数 IOR 的 Fresnel 方程（支持正入射和任意角度）
- `parseSpectralData()` - 解析光谱数据文件（波长-n-k 格式）

## 关键实现逻辑
1. 对 CIE 2006 10 度色彩匹配函数（390-830nm, 441 个采样）逐波长积分
2. 每个波长处计算复数 IOR 的 Fresnel 反射率，乘以 CMF 和 D65 光源 SPD
3. 超出 sRGB 色域的颜色按最大分量缩放（如金色）
4. f82 角度（默认 81.7 度）用于 Lazanyi 误差项，改善金属 Fresnel 近似
5. 光谱数据文件格式：先列出所有波长的 n 值，再列出所有波长的 k 值

## 依赖关系
- `math/` - 复数运算和矩阵变换
- CIE 2006 色彩匹配函数和 D65 光源数据

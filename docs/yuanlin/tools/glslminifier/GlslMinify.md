# GlslMinify.h / GlslMinify.cpp

## 文件概述
GLSL 着色器代码压缩库。通过正则表达式去除注释、空行和缩进来缩小着色器文件体积。

## 核心函数
- `minifyGlsl()` - 对 GLSL 字符串执行压缩，支持通过选项标志控制压缩级别

## 核心枚举
- `GlslMinifyOptions` - 位标志枚举
  - `STRIP_COMMENTS` (0x1) - 去除 `//` 和 `/* */` 注释
  - `STRIP_EMPTY_LINES` (0x2) - 合并连续空行为单行
  - `STRIP_INDENTATION` (0x4) - 去除行首缩进
  - `ALL` (0xFFFFFFFF) - 启用所有优化

## 关键实现逻辑
1. 使用预编译的静态正则表达式（C++14 `std::regex`）
2. 注释去除分两步：先处理 `//` 单行注释，再处理 `/* */` 多行注释（非贪婪匹配）
3. 空行合并使用捕获组 `(\n|\r\n)+`，替换为单个换行符（`$1`）
4. 缩进去除保留行首的换行符或文件开头位置

## 依赖关系
- 仅依赖 C++ 标准库 `<regex>`
